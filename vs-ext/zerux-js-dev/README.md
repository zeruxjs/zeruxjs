# ZeruxJS Dev Tools

The official Visual Studio Code extension for ZeruxJS development.

## 🚀 Key Features

| Feature | Description |
| :--- | :--- |
| **Intellisense** | Intelligent code completion for framework-specific APIs. |
| **Snippets** | Quick-start code snippets for common patterns and components. |
| **Error Detection** | Real-time linting and framework-specific error highlighting. |
| **Dev Tools** | Integrated debugging and performance monitoring support. |
| **Short Keys** | Auto-completion and quick actions for faster development. |
